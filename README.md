# 🏠 Housing Conditions Analysis

Academic project developed in Python to analyze housing conditions, family income, and demographic indicators.

## Features

- Family registration
- Population analysis by neighborhood
- Average income in Reunidas neighborhood
- Percentage of parents with higher education
- Average number of people per family
- Percentage of families without sewage access
- Number of apartments in Berger neighborhood

## Technologies

- Python 

## Concepts Practiced

- Variables
- User input
- Conditional statements (`if`, `else`)
- Loops (`while`)
- Counters and accumulators
- Statistical calculations
